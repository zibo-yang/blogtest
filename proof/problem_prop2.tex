PROBLEM (Proposition 2): For every a in (0, 3/2), define C_a = -a/2 + a^2/6 and P(a) = 2*integral_0^a dv/sqrt(v(a-v)(1-(a+v)/3)). Prove there exists a nonzero function u_a in C^3([0,P(a)]) such that u_a'' + u_a + (1/2)u_a^2 = C_a, with u_a(0) = u_a'(0) = u_a(P(a)) = u_a'(P(a)) = 0, and min u_a = -a.
